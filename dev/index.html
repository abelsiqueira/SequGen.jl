SequGen.SineWave — Type
SineWave(amplitude, period, phase)
SineWave(:from_phase, amplitude, period, phase)

Creates a sine wave, i.e., something of the form

ϕ(t) = A sin( 2π t / T + θ₀),

where A is the amplitude, T is the period, and θ₀ is the phase.

SequGen.StepwiseGaussianNoise — Type
StepwiseGaussianNoise(stepsize; σ = 1.0)

Returns a Gaussian noise with standard deviation σ and the given stepsize. The noise is constant between interval of size stepsize. For instance,

sample(StepwiseGaussianNoise(1.0), 0.0, 2.0, 10)

will return a sample of 10 elements with 5 equal values followed by other 5 equal values. This is equivalent to [randn() * ones(5); randn() * ones(5)] in this case.
